# CI/CD Pipeline Project Report

## Project Title
Automated CI/CD Pipeline using GitHub and Jenkins

## Objective
To automate the deployment process of a simple website using CI/CD pipeline.

## Methodology
1. Source code is pushed to GitHub repository.
2. Jenkins pipeline pulls the latest code.
3. Jenkins performs build and test steps.
4. Jenkins runs a Python server to host the website.
5. A local URL is generated to view the output in browser.

## Result
The website is successfully deployed and accessible through a local URL.

## Conclusion
The CI/CD pipeline automates deployment and reduces human errors.
